## Switching keyboard language with a double alt
This AutoHotkey v2 script always stays in memory and toggles the input language with a double press of the left Alt key.
In the same way, you can set up toggling with a double press of Ctrl, Shift, or any other key combination.

You can run this script by clicking on its icon. Using AutoHotkey, you can generate a regular exe file from the script and place it in the startup folder. This will result in a standard resident program that automatically runs in the background without any messages.
